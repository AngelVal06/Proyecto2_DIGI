Resumen del Proyecto

Advanced Text Formatter es una aplicación completa para formatear texto con interfaz gráfica y capacidad de procesamiento por lotes.

Características Principales

Interfaz Gráfica

Interfaz intuitiva con TKinter que permite formatear texto fácilmente.

Múltiples Formatos

Soporte para 10+ operaciones de formateo diferentes.

Procesamiento por Lotes

Capacidad de procesar múltiples archivos (.txt, .csv) a la vez.

Drag and Drop

Soporte para arrastrar y soltar archivos directamente a la aplicación.

Instalación y Configuración

Requisitos

  • Python 3.8 o superior
  • Bibliotecas: tkinter, tkinterdnd2, re, os

Instalación

pip install tkinterdnd2

Ejecución

python Proyecto.py

Clase Principal: TextFormatterApp

Clase que representa la aplicación principal, hereda de TkinterDnD.Tk para soportar drag and drop.

Constructor de la clase. Configura la ventana principal y los estilos de la interfaz.

def __init__(self):
    super().__init__()
    self.title("Advanced Text Formatter")
    self.geometry("700x800")
    self.configure(bg="#f0f0f0")
    self.resizable(True, True)
    
    # Configuración de estilos
    self.style = ttk.Style()
    self.style.configure('TFrame', background='#f0f0f0')
    self.style.configure('TLabel', background='#f0f0f0', font=('Arial', 10))
    self.style.configure('TButton', font=('Arial', 9), padding=5)
    
    self.create_widgets()

Funciones de Formateo de Texto

convertir_mayusculas(self)

Convierte el texto de entrada a mayúsculas.

Ejemplo

Entrada: "hola mundo" → Salida: "HOLA MUNDO"

convertir_minusculas(self)

Convierte el texto de entrada a minúsculas.

formato_titulo(self)

Convierte el texto a formato de título (capitaliza cada palabra).

formato_oracion(self)

Capitaliza solo la primera letra de la primera palabra.

formato_snake_case(self)

Convierte el texto a snake_case (todo en minúsculas con guiones bajos).

Ejemplo

Entrada: "Texto de prueba" → Salida: "texto_de_prueba"

texto = re.sub(r'[^a-zA-Z0-9]+', '_', texto).lower()
formato_camel_case(self)

Convierte el texto a camelCase (primera palabra en minúsculas, siguientes capitalizadas).

Ejemplo

Entrada: "texto ejemplo" → Salida: "textoEjemplo"

texto = re.sub(r'[^a-zA-Z0-9]+', ' ', texto).split()
texto = texto[0].lower() + ''.join(word.capitalize() for word in texto[1:])
quitar_espacios(self)

Elimina espacios extras, dejando solo un espacio entre palabras.

quitar_caracteres_especiales(self)

Elimina todos los caracteres no alfanuméricos (excepto espacios).

quitar_duplicados(self)

Elimina palabras duplicadas manteniendo el orden original.

generar_markdown(self)

Genera un encabezado Markdown nivel 1 con el texto.

Ejemplo

Entrada: "Título" → Salida: "# Título"

generar_html(self)

Genera un encabezado HTML h1 con el texto.

Ejemplo

Entrada: "Título" → Salida: "<h1>Título</h1>"

copiar_resultado(self)

Copia el texto formateado al portapapeles del sistema.

Funciones de Procesamiento por Lotes

on_drop(self, event)

Maneja el evento de arrastrar y soltar archivos.

Argumentos

event: Evento de drop con información de los archivos.

Comportamiento

Solo acepta archivos .txt o .csv y los añade a la lista de procesamiento.

select_files(self)

Abre un diálogo para seleccionar archivos a procesar.

clear_file_list(self)

Limpia la lista de archivos seleccionados.

process_files(self)

Procesa todos los archivos seleccionados con el formato elegido.

Flujo de trabajo

  1. Verifica que hay archivos seleccionados
  2. Obtiene la función de formateo correspondiente
  3. Solicita directorio de salida
  4. Procesa cada archivo individualmente
  5. Guarda los resultados con prefijo "processed_"
  6. Muestra resumen al finalizar

Ejemplos de Uso

Formateo Básico

# Convertir a mayúsculas
app = TextFormatterApp()
app.text_input.insert("1.0", "texto de ejemplo")
app.convertir_mayusculas()
print(app.text_output.get("1.0", "end-1c"))  # "TEXTO DE EJEMPLO"

Procesamiento por Lotes

# Procesar múltiples archivos
app = TextFormatterApp()
# Simular drag and drop de archivos
app.file_listbox.insert(tk.END, "archivo1.txt")
app.file_listbox.insert(tk.END, "archivo2.csv")
# Seleccionar formato
app.format_var.set("MAYÚSCULAS")
# Procesar
app.process_files()
